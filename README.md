# PayZero 💸

Instant crypto payments with zero hassle. Built on Base Sepolia testnet.

## Features
- 🔗 Connect MetaMask wallet
- 💰 Send ETH & USDC instantly
- 📱 QR code payments
- ⚡ Near-zero gas fees
- 🔒 Secure blockchain transactions

## Get Started
1. Install [MetaMask](https://metamask.io/)
2. Get Base Sepolia testnet tokens from [faucet](https://www.alchemy.com/faucets/base-sepolia)
3. Connect wallet and start sending payments

## Tech Stack
- React
- Web3 / Ethereum
- Base Sepolia L2
- MetaMask integration
